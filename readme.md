### Технологии
- React
- Typescript
- MobX

### Установка
- клонирование приложения командой
  ```bsh
  git clone https://github.com/PotapenkoVadim/kanban-app.git
- установка зависимостей
  ```js
  npm install
- запуск локального сервера port:4200
  ```js
  npm run start
